API Reference

This page renders the function docstrings for the public API.

For mathematical definitions and worked usage examples, see Math and Usage.

Core Evaluation

SpheroidalWaveFunctions.smn — Function

Compute angular spheroidal wave functions and derivatives with respect to eta.

Use this for batched angular evaluations for real or complex c.

source
SpheroidalWaveFunctions.rmn — Function

Compute radial spheroidal wave functions and derivatives with respect to x.

Supports radial kinds 1-4 and returns complex-valued outputs.

source

Analysis and Diagnostics

SpheroidalWaveFunctions.eigenvalue — Function

Compute the spheroidal separation constant lambda_mn(c).

SpheroidalWaveFunctions.accuracy — Function
